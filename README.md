# Web Summer Camp 2026 — Workshop Playgrounds

My hands-on outputs from the workshops I attended at [Web Summer Camp 2026](https://websummercamp.com/2026)
(July 2–4, 2026, Opatija, Croatia — AI Engineering track).

Each folder is the working state of my machine at the end of a workshop: the
facilitator's material plus whatever I built, changed, or broke along the way.
Every folder's README links back to the session page and the original workshop
repository.
